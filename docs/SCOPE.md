# Scope

## V1 Includes
- Local web app with FastAPI backend and React TypeScript frontend.
- One active script per project.
- Multiple jobs across projects, with one active generation job per project.
- In-process job queue and worker pool.
- Configurable concurrency for fake TTS, Chatterbox TTS, and WAV merge.
- WAV output.
- SQLite metadata persistence.
- Filesystem storage under a configured local storage root.
- Fake deterministic TTS for all normal tests.
- Lazy Chatterbox adapter.
